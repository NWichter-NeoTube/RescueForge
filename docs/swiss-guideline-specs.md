# FKS-Richtlinie Orientierungspläne - Technische Spezifikation

**Quelle:** Feuerwehr Koordination Schweiz (FKS), gültig ab 01.08.2024
**Referenz:** https://docs.feukos.ch/richtlinie-orientierungsplane/

## Plantypen

### Deckblatt (Cover Sheet)
- Objektdefinition: Gebäudenutzung + Adresse
- Legende mit allen Symbolen
- Situationsplan mit Gebäudeseiten-Bezeichnung
- Titelblock: Plantitel, Objekt, Adresse, Datum

### Geschosspläne (Floor Plans)
- Minimum 1 Plan pro Geschoss
- Format: **A4 oder A3**
- Alle Pläne im **gleichen Massstab (max. 1:300)**
- Alle Pläne in **gleicher Ausrichtung**
- Bei Aufteilung: Gesamtübersicht auf jedem Teilplan
- Geschosslage-Querschnitt (zeigt alle Stockwerke)

### Situationsplan
- Gebäude im Kontext (Strassen, Landmarks)
- Gebäudeseiten-Bezeichnung (See, Autobahn, etc.)
- **Nordpfeil** erforderlich
- BMA-BS Standort (rot)
- Sprinkler-Zugang (blau)

## Architektur-Elemente

### Was muss gezeigt werden
- **Wände:** Schwarze Linien (dick für tragend, standard für nicht-tragend)
- **Türen:** Standard Plandarstellung (Türschwung sichtbar)
- **Treppen:** Standard Plandarstellung
- **Raumbeschriftungen:** Müssen mit BMA-Programmierung übereinstimmen
- **Gefährliche Räume:** Rot hervorgehoben (z.B. Lösungsmittel, Heizung)
- **Grosse Räume:** Fläche (A) und Volumen (V) angeben

### Was NICHT gezeigt werden soll
- Möbel
- Vermassung
- Technische Spezifikationen
- Unnötige Details

## Farbgebung

### Fluchtwege
| Element | Farbe |
|---|---|
| Vertikale Fluchtwege (Treppenhäuser) | **Dunkelgrün** (gefüllt) |
| Horizontale Fluchtwege (Korridore) | **Hellgrün** (gefüllt) |

### Brandmeldeanlage (BMA)
| Element | Farbe |
|---|---|
| Meldergruppen-Grenzen | **Einzigartige Farbe pro Gruppe** |
| Handfeuermelder (HFM) | **Immer Rot** |
| BMA-Bedienstelle | **Rot** (Hintergrund) |
| BMA-Zentrale | **Rot** (Hintergrund) |
| Gefährliche Räume Text | **Rot** |

### Sprinkleranlage (SPA)
| Element | Farbe |
|---|---|
| Sprinkler-geschützte Zonen | **Blau/Cyan, schraffiert** |
| Sprinkler-Gruppen-Grenzen | **Cyan/Hellblau** |
| Zugang Sprinklerzentrale | **Blauer Pfeil** |

### Zugang
| Element | Farbe |
|---|---|
| Hauptzugang Feuerwehr | **Roter Pfeil** (gross, gefüllt) |
| Zugang Sprinklerzentrale | **Blauer Pfeil** (gross, gefüllt) |
| Schlüsseldepot | Rote Box mit Schlüssel-Symbol |

## Symbole (nach SES-Richtlinie)

### Melder
- Rauchmelder: "S" im Quadrat
- Rauchmelder Hohlboden: "S" mit Punkt unten
- Rauchmelder Hohldecke: "S" mit Punkt oben
- Handfeuermelder: Rotes Dreieck
- Gasmelder: "CO" im Kreis

### Alarmgeräte
- Akustisch: Rotes Rechteck mit Glocke
- Optisch/Akustisch: Kombiniertes Symbol
- Optisch: Rotes Rechteck mit Blitz

### Mindestgrössen
- Melder-Symbole: **mindestens 3x3 mm**
- Alle Symbole an tatsächlichem Montageort

## Format-Zusammenfassung

| Eigenschaft | Wert |
|---|---|
| Papierformat | A4 oder A3 |
| Max. Massstab | 1:300 |
| Auslieferung | Primär elektronisch |
| Aktualisierung | Bei jeder Gebäude-/Anlagenänderung |
